# OS Lecture 系统化笔记目录

本目录对应 `OS` 文件夹内 12 份 lecture PDF。  
整理原则如下：

- 每个 lecture 单独一份 Markdown。
- 每份笔记按“本讲主线 -> 系统知识点 -> 例子 -> 本讲结论”的方式组织。
- 目标不是逐页转写，而是把每讲内容重组为更适合复习和建立知识框架的讲义。

## 推荐阅读顺序

1. 先读 `01` 到 `03`，建立总视角、接口和底层 ISA 概念。
2. 再读 `04` 到 `08`，把进程、线程和线程实现连成一条线。
3. 然后读 `09` 到 `10`，形成完整调度视角。
4. 最后读 `11` 到 `12`，集中解决并发正确性问题。

- `markdown_summaries` 目录更偏向逐页抽取、保留原讲义信息。
- 当前目录更偏向系统化整理，适合复习、串联和讲解。
